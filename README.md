# PCYOAG
This is my Python - Create Your Own Adventure Game Project. 

Five Days in Vegas is a text-based adventure game written in Python. 

The player has a sudden layover and experiences five days in Las Vegas with choices, randomness, and multiple endings.


## How to Run
Steps:
1. Download or clone this repository
2. Make sure `main.py` and `backbone.py` are in the same folder
3. Run the game with:
python main.py

Note: terminal used for testing was https://www.online-python.com/
